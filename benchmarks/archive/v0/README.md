# BrouwerBench v0 Archive

This directory preserves the original 36-task BrouwerBench pilot for reproducibility.

The current benchmark and all reported paper results use `benchmarks/data/brouwerbench_v1.jsonl`.
Treat v0 as historical material only.

Contents:

- `data/`: the original v0 JSONL dataset.
- `results/`: pilot raw outputs, scored outputs, reports, comparison notes, and LaTeX tables.
- `scores/`: manual score files used for the v0 pilot reports.
